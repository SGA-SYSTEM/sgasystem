Recursos & Tecnologia



JavaScript


JavaScript é uma linguagem de programação interpretada2. Originalmente implementada como parte dos navegadores web para que scripts pudessem ser executados do lado do cliente e interagissem com o usuário sem a necessidade deste script passar pelo servidor, realizando comunicação assíncrona e alterando o conteúdo do documento.



Praticidade


A modalidade online permite superar as desvantagens e constrangimentos do formato papel. A grande parte dos cursos online permite que os alunos tenham o seu próprio horário para estudar e realizar as provas.



Python


Python é uma linguagem de programação de alto nível4 , interpretada, imperativa, orientada a objetos, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991. Atualmente possui um modelo de desenvolvimento comunitário, aberto e gerenciado pela organização sem fins lucrativos Python Software Foundation.



Django


Django é um framework para desenvolvimento rápido para web, escrito em Python, que utiliza o padrão MTV (model - template - view). Foi criado originalmente como sistema para gerenciar um site jornalístico na cidade de Lawrence, no Kansas. Tornou-se um projeto de código aberto e foi publicado sob a licença BSD em 2005.

Sobre o projeto

O software criado tem como objetivo principal impulsionar o método de avaliação tradicional de todos os ângulos do usuário (docente e aluno), onde, através desse é possível aplicar provas e acompanhar o desenvolvimento unitário e geral dos alunos, tornando mais rápido todo o processo burocrático que existe entre elaboração, aplicação, resolução e correção de provas. O aluno contará com a facilidade de ter a nota da avaliação de forma instantânea, não precisando esperar ate que seja corrigida e evitando que o docente perca horas corrigindo as avaliações.
